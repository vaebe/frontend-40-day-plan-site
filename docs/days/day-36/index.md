# 第 36 天：JS + TS 高频面试复盘

::: tip 阶段
**阶段：** 面试 / 简历 / 投递

**用法：** 每个时间段都已经把“具体看哪些章节”和“学完怎么验收”放在一起。先看验收标准，再照着具体安排做。
:::

[[toc]]

## 08:30-09:00 晨间复盘

::: tip 这一段学完要达到
- 知道：今天不学新知识，做表达训练。
- 理解：会写和会讲是两种能力。
- 学会：打开所有 JS/TS 笔记。
- 验收：列出要复盘的 30 个问题。
:::

### 具体安排

- 打开 JS 和 TS 笔记。
- 写今天目标：把高频题变成自己的话。


## 09:00-12:00 上午学习

::: tip 这一段学完要达到
- 知道：闭包、原型链、Promise、事件循环是 JS 核心。
- 理解：每个问题都要能配例子。
- 学会：口头回答基础题。
- 验收：录音回答 4 个问题，每个 1 分钟左右。
:::

### 具体安排

- 复看闭包：https://developer.mozilla.org/zh-CN/docs/Web/JavaScript/Guide/Closures
- 复看原型链：https://developer.mozilla.org/zh-CN/docs/Web/JavaScript/Inheritance_and_the_prototype_chain
- 复看 Promise：https://developer.mozilla.org/zh-CN/docs/Web/JavaScript/Reference/Global_Objects/Promise
- 复看事件循环：https://developer.mozilla.org/zh-CN/docs/Web/JavaScript/Event_loop


## 12:00-14:00 中午轻复盘

::: tip 这一段学完要达到
- 知道：卡壳点就是要补的点。
- 理解：听自己录音能发现表达问题。
- 学会：标记讲不顺的问题。
- 验收：至少记录 5 个卡壳点。
:::

### 具体安排

- 录音回答：闭包、原型链、Promise、事件循环。


## 14:00-18:00 下午项目

::: tip 这一段学完要达到
- 知道：TS 面试常问 interface/type、泛型、联合类型、工具类型。
- 理解：要结合你的项目接口类型讲。
- 学会：整理 30 问 30 答。
- 验收：每个答案至少有一句项目例子。
:::

### 具体安排

- 复看 TS Everyday Types、Narrowing、Generics、Utility Types。
  - 必看章节：The primitives、Arrays、any、Type Annotations on Variables、Functions、Object Types、Union Types、Type Aliases、Interfaces、Literal Types、null and undefined。
  - 读完要会：能给变量、数组、函数、对象补基础类型，尽量不写 `any`。
- 整理 30 个 JS + TS 高频问答。


## 19:30-21:00 晚上算法

::: tip 这一段学完要达到
- 知道：薄弱题型要集中练。
- 理解：随机刷不如定向补短板。
- 学会：刷 3 道薄弱题。
- 验收：每题写出错点或关键思路。
:::

### 具体安排

- 算法薄弱题型 3 题。


## 21:00-22:30 夜间输出

::: tip 这一段学完要达到
- 知道：面试笔记要短而能讲。
- 理解：不要抄长篇八股。
- 学会：写自答版。
- 验收：完成《JS + TS 高频面试自答版》。
:::

### 具体安排

- 写《JS + TS 高频面试自答版》。
- 当天产出：30 问 30 答。
