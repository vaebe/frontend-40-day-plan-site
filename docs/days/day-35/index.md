## 学习结果和验收

08:30-09:00：

- 知道：今天只是了解 React，不转移主线。
- 理解：大厂可能问 React，但你主项目仍然是 Vue。
- 学会：定好边界。
- 验收：写下“今天不重做项目”。

09:00-12:00：

- 知道：React 有 JSX、props、state、组件。
- 理解：React 更强调用 JS 描述 UI。
- 学会：看懂简单 React 组件。
- 验收：能写一个最小 React 组件。

12:00-14:00：

- 知道：React 和 Vue 都是组件化框架。
- 理解：两者状态更新和模板表达方式不同。
- 学会：写基础对比。
- 验收：列出 5 条 Vue/React 差异。

14:00-18:00：

- 知道：React demo 可选，不要影响主项目。
- 理解：现阶段目标是“不怯场”，不是双框架精通。
- 学会：写小 demo 或整理笔记。
- 验收：能看懂 useState 基础例子。

19:30-21:00：

- 知道：算法不能断。
- 理解：错题复盘要持续。
- 学会：补薄弱题型。
- 验收：2 道错题复盘完成。

21:00-22:30：

- 知道：面试中可以坦诚主 Vue，了解 React。
- 理解：迁移能力比浅尝一堆框架更重要。
- 学会：写对比笔记。
- 验收：写完《Vue 和 React 基础对比》。
