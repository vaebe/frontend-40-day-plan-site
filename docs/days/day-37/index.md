## 今日安排

08:30-09:00：

- 写今天目标：Vue 问题能讲项目例子。

09:00-12:00：

- 复看 Vue 响应式基础：https://cn.vuejs.org/guide/essentials/reactivity-fundamentals.html
- 复看 computed：https://cn.vuejs.org/guide/essentials/computed.html
- 复看 watchers：https://cn.vuejs.org/guide/essentials/watchers.html
- 复看组件通信、插槽、Router、Pinia。

12:00-14:00：

- 录音回答：ref/reactive、computed/watch、组件通信。

14:00-18:00：

- 整理 30 个 Vue 高频问答。
- 每个回答尽量结合你的项目。

19:30-21:00：

- 算法薄弱题型 3 题。

21:00-22:30：

- 写《Vue 高频面试自答版》。
- 当天产出：30 个 Vue 问答。

## 学习结果和验收

08:30-09:00：

- 知道：Vue 回答要尽量贴项目。
- 理解：只背 API 会显得像没做过项目。
- 学会：把项目例子和问题配对。
- 验收：列出 10 个可结合项目讲的 Vue 点。

09:00-12:00：

- 知道：响应式、computed/watch、组件通信、生命周期。
- 理解：Vue 面试核心是状态变化如何驱动 UI。
- 学会：口头回答 API 区别。
- 验收：ref/reactive、computed/watch 能讲清。

12:00-14:00：

- 知道：录音是练表达的好方法。
- 理解：面试不是写作文，要自然讲出来。
- 学会：录音复述。
- 验收：至少录 3 个 Vue 问题。

14:00-18:00：

- 知道：Router、Pinia、组件封装、权限都能从项目展开。
- 理解：项目面和 Vue 面会交叉。
- 学会：整理 30 个 Vue 问答。
- 验收：每个回答尽量带项目例子。

19:30-21:00：

- 知道：算法继续维持手感。
- 理解：面试前不能完全断代码。
- 学会：补薄弱题。
- 验收：3 道题完成复盘。

21:00-22:30：

- 知道：Vue 高频题要形成自己的话术。
- 理解：答案不是越长越好。
- 学会：写自答版。
- 验收：完成《Vue 高频面试自答版》。
