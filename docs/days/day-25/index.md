## 09:00-12:00 上午学习

::: tip 这一段学完要达到

- 学会：写作用域插槽。
- 验收：能写 `&lt;slot :row="row"&gt;` 这类结构。
:::

### 具体安排

- 看 Vue 插槽：https://cn.vuejs.org/guide/components/slots.html
  - 必看章节：插槽内容与出口、默认内容、具名插槽、条件插槽、作用域插槽。
  - 读完要会：知道表格操作列、弹窗 footer 为什么常用插槽。
- 看 作用域插槽：https://cn.vuejs.org/guide/components/slots.html#scoped-slots
  - 只看 `作用域插槽` 小节和具名作用域插槽示例。
  - 读完要会：能解释父组件如何拿到子组件暴露出来的行数据。
- 看 Element Plus Table 插槽示例。
  - 必看章节：基础表格、带斑马纹、固定表头、多选、排序、自定义列模板。
  - 读完要会：能渲染用户列表和操作按钮列。
